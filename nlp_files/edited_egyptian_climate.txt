<span class="impulse-fixation">In</span> the <span class="impulse-fixation">fourth</span> <span class="impulse-fixation">century,</span> as the <span class="impulse-fixation">Roman</span> <span class="impulse-fixation">Empire</span> <span class="impulse-fixation">divided,</span> <span class="impulse-fixation">Egypt</span> <span class="impulse-fixation">found</span> itself in the <span class="impulse-fixation">Eastern</span> <span class="impulse-fixation">Empire</span> with its <span class="impulse-fixation">capital</span> at <span class="impulse-fixation">Constantinople.</span> <span class="impulse-fixation">In</span> the <span class="impulse-fixation">waning</span> <span class="impulse-fixation">years</span> of the <span class="impulse-fixation">Empire,</span> <span class="impulse-fixation">Egypt</span> <span class="impulse-fixation">fell</span> to the <span class="impulse-fixation">Sasanian</span> <span class="impulse-fixation">Persian</span> <span class="impulse-fixation">army</span> in the <span class="impulse-fixation">Sasanian</span> <span class="impulse-fixation">conquest</span> of <span class="impulse-fixation">Egypt</span> <span class="impulse-fixation">(618–628).</span> <span class="impulse-fixation">It</span> was then <span class="impulse-fixation">recaptured</span> by the <span class="impulse-fixation">Byzantine</span> <span class="impulse-fixation">emperor</span> <span class="impulse-fixation">Heraclius</span> <span class="impulse-fixation">(629–639),</span> and was <span class="impulse-fixation">finally</span> <span class="impulse-fixation">captured</span> by <span class="impulse-fixation">Muslim</span> <span class="impulse-fixation">Rashidun</span> <span class="impulse-fixation">army</span> in <span class="impulse-fixation">639–641,</span> <span class="impulse-fixation">ending</span> <span class="impulse-fixation">Byzantine</span> <span class="impulse-fixation">rule.</span>
<span class="impulse-fixation"><h2>Government</span> and <span class="impulse-fixation">economy</h2></span>
<span class="impulse-fixation"><h3>Administration</span> and <span class="impulse-fixation">commerce</h3></span>
<span class="impulse-fixation">The</span> <span class="impulse-fixation">pharaoh</span> was the <span class="impulse-fixation">absolute</span> <span class="impulse-fixation">monarch</span> of the <span class="impulse-fixation">country</span> <span class="impulse-fixation">and,</span> at <span class="impulse-fixation">least</span> in <span class="impulse-fixation">theory,</span> <span <span class="impulse-fixation">Not</span> only were they <span class="impulse-fixation">places</span> of <span class="impulse-fixation">worship,</span> but were <span <span class="impulse-fixation">Much</span> of the <span class="impulse-fixation">economy</span> was <span class="impulse-fixation">centrally</span> <span class="impulse-fixation">organized</span> and class="impulse-fixation">laborer</span> <span class="impulse-fixation">might</span> <span class="impulse-fixation">earn</span> <span class="impulse-fixation">5+1⁄2</span> <span <span class="impulse-fixation"><h3>Agriculture</h3></span>
<span class="impulse-fixation">A</span> <span class="impulse-fixation">combination</span> of <span class="impulse-fixation">favorable</span> <span class="impulse-fixation">geographical</span> <span <span class="impulse-fixation">Farming</span> in <span class="impulse-fixation">Egypt</span> was <span class="impulse-fixation">dependent</span> on the <span class="impulse-fixation">cycle</span> of <span class="impulse-fixation">Egypt</span> <span class="impulse-fixation">received</span> <span class="impulse-fixation">little</span> <span class="impulse-fixation">rainfall,</span> so <span <span class="impulse-fixation">The</span> <span class="impulse-fixation">ancient</span> <span class="impulse-fixation">Egyptians</span> <span class="impulse-fixation">cultivated</span> <span <span class="impulse-fixation">Egyptian</span> <span class="impulse-fixation">society</span> was <span class="impulse-fixation">highly</span> <span class="impulse-fixation">stratified,</span> and <span class="impulse-fixation">state</span> <span class="impulse-fixation">control,</span> <span class="impulse-fixation">working</span> in the <span class="impulse-fixation">shops</span> <span
